Features and Potential Problems

  • Select / Insert / Search / Update / Delete implemented for all entities except Mediphar (composite table)
  • Mediphar has Select / Insert / Delete functionality, but is missing Update and Search functionality as these seem harder to implement with a composite table (and also it's not required to have Update and Search on every entity)
  • Input validation is mostly supported. Currently you're allowed to enter numbers into string inputs, as some input fields such as "Email" might contain numbers. Thinking about how to improve the input validation for the future.
  • Invalid inputs will result in a warning showing, although this warning won't go away until a correct input is entered.
  • Make sure you actually click on the Button prompts on the site to navigate. Pressing "Enter" key is currently disabled due to this leading to wrong URL's

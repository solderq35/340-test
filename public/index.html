General Specifications

  • Select / Insert / Search / Update / Delete implemented for all entities except Mediphar (Intersection Table using composite primary key).
  • Mediphar has Select / Insert / Delete functionality, but is missing Update and Search functionality as these seem harder to implement with composite primary key (and also it's not required to have Update and Search on every entity).
  • Note: Pressing the "Enter" key (such as after typing in a search term) is currently not supported. You need to actually click the "Search / Update / Delete / Submit" buttons.
